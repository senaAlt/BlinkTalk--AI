# BlinkTalk-AI 

**TR:** Göz kırpma hareketleriyle iletişim kurmayı sağlayan yapay zeka destekli sistem. Konuşma güçlüğü çeken bireyler için geliştirilmiştir.

**EN:** An AI-powered communication system that enables interaction through eye blink gestures. Developed for individuals with speech difficulties.

---

##  Nasıl Çalışır? / How It Works
**TR:** Kamera aracılığıyla göz kırpma hareketleri tespit edilir. Kısa (K) ve uzun (U) kırpmalar kombinasyonuyla komutlar oluşturulur.

**EN:** Eye blink movements are detected via camera. Commands are created using combinations of short (K) and long (U) blinks.

---

# Komutlar / Commands
| Hareket / Gesture | Komut / Command |
|-------------------|-----------------|
| K | Su istiyorum / I need water |
| K - K | Yardım çağırın! / Call for help! |
| K - K - K | Teşekkür ederim / Thank you |
| U | Evet / Yes |
| U - U | Hayır / No |
| K - U | Ağrım var / I'm in pain |
